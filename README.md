# Base Project

Workspace gom 2 base project:

- `frontend`: React + Vite
- `backend`: Spring Boot

## Backend

Can Maven hoac Gradle/IDE co ho tro Spring Boot.

```powershell
cd backend
mvn spring-boot:run
```

Backend mac dinh: `http://localhost:8080`

## Frontend

```powershell
cd frontend
npm.cmd install
npm.cmd run dev
```

Frontend mac dinh: `http://localhost:5173`
